# 项目功能介绍
  整个项目前端采用Vue-cli脚手架搭建，后端由Django开放接口，实现前后端分离开发
# 项目功能：
  * 负责整个项目的构建与开发，使用vue-cli搭建前端开发环境
  * jwt进行会话控制
  * 邮箱进行注册账号的激活
  * 第三方微信，qq进行登录
  * 沙箱支付模拟真实支付场景
  * 乐观锁与悲观锁解决商品并发问题
  * scrapy爬取不同的增量/全量数据
  * 验证码方式进行登录验证
  * axios进行数据请求
  * 第三方sm.ms图床保存用户上传图片
  * vue router实现路由切换以及访问权限控制
